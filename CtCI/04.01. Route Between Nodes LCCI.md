# Route Between Nodes LCCI

[[bfs]] [[dfs]]

Given a directed graph, design an algorithm to find out whether there is a route between two nodes.

Example1:

 Input: n = 3, graph = [[0, 1], [0, 2], [1, 2], [1, 2]], start = 0, target = 2
 Output: true

Example2:

 Input: n = 5, graph = [[0, 1], [0, 2], [0, 4], [0, 4], [0, 1], [1, 3], [1, 4], [1, 3], [2, 3], [3, 4]], start = 0, target = 4
 Output true

Note:

    0 <= n <= 100000
    All node numbers are within the range [0, n].
    There might be self cycles and duplicated edges.

bfs:

```python
class Solution:
    def findWhetherExistsPath(self, n: int, graph: List[List[int]], start: int, target: int) -> bool:
        if start == target:
            return True

        d = defaultdict(set)
        for s, e in graph:
            d[s].add(e)

        visited = {start}
        dq = deque([start])

        while dq:
            node = dq.popleft()
            for i in d[node]:
                if i == target:        # 命中立即返回，不再继续入队
                    return True
                if i not in visited:
                    visited.add(i)
                    dq.append(i)

        return False
```

双向bfs：

```python
class Solution:
    def findWhetherExistsPath(self, n: int, graph: List[List[int]], start: int, target: int) -> bool:
        if start == target:
            return True

        # 正向 + 反向邻接表
        fwd = defaultdict(set)
        bwd = defaultdict(set)
        for s, e in graph:
            fwd[s].add(e)
            bwd[e].add(s)

        # 两侧各自的 visited 集合（同时充当队列的"下一层"）
        front, back = {start}, {target}
        visited_f, visited_b = {start}, {target}

        while front and back:
            # 每次展开较小的那一侧，减少不必要扩展
            if len(front) > len(back):
                front, back = back, front
                fwd, bwd = bwd, fwd
                visited_f, visited_b = visited_b, visited_f

            next_front = set()
            for node in front:
                for nei in fwd[node]:
                    if nei in visited_b:   # 两侧相遇
                        return True
                    if nei not in visited_f:
                        visited_f.add(nei)
                        next_front.add(nei)
            front = next_front

        return False
```

                if i not in visited:

recursive dfs:

```python
class Solution:
    def findWhetherExistsPath(self, n: int, graph: List[List[int]], start: int, target: int) -> bool:
        if start == target:
            return True

        d = defaultdict(set)
        for s, e in graph:
            d[s].add(e)

        visited = {start}
        return self.dfs(d, visited, start, target)

    def dfs(self, d, visited, node, target):
        for i in d[node]:
            if i == target:        # 命中立即返回，不再继续入队
                return True
            if i not in visited:
                visited.add(i)
                if self.dfs(d, visited, i, target):
                    return True
        return False
```
